# 03 Data Cleaning

This step creates the first processed datasets.

The main script is:

```bash
.venv/bin/python src/clean_data.py
```

Questions to answer after running the script:

1. How many fights have a clear winner?
2. How many fights are draws or no contests?
3. Which fighter attributes have missing values?
4. Which cleaned columns might be useful for later feature engineering?

No model training happens in this step.
